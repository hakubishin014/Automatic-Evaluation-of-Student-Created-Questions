1. 【因数分解を用いる基本問題】  
   次の二次方程式を因数分解して解け。  
   2x² − 5x − 3 = 0

2. 【平方完成を用いる問題】  
   x² + 6x − 7 = 0 を平方完成して解け。

3. 【解の公式を用いる問題】  
   3x² + x − 4 = 0 の解を解の公式で求めよ。

4. 【判別式による実数解の個数】  
   次の二次方程式の判別式 Δ を求め、実数解の個数を答えよ。  
   5x² − 2kx + 1 = 0  
   （ただし k は定数とする）

5. 【与えられた解から方程式をつくる】  
   ある二次方程式の解が x = 2, x = − 3 であるとき、その方程式を標準形 ax²+bx+c=0 で表せ。

6. 【長さの関係から】  
   長さ  x (m) の辺をもつ正方形の面積が、長さ (x−1) (m) の辺をもつ正方形の面積の 3 倍になるとき、x を求めよ。

7. 【仕事算の応用】  
   A さんが単独で仕事をすると 6 日かかり、B さんだけだと 8 日かかる。ふたりが同時に作業を始め、x 日後に全体の仕事が終わったとき、x を求めよ。

8. 【最大・最小問題】  
   関数 y = −x² + 4x + 5 の最大値を求め、そのときの x の値を答えよ。

9. 【速度・距離の問題】  
   ある物体が斜め上方に投げ上げられたときの高さ h (m) は時間 t (s) に対して  
   h = −5t² + 20t + 2  
   で与えられる。この物体が地面に落ちるまでの時間を求めよ。

10. 【実数範囲の利用】  
    二次方程式 x² + px + 4 = 0 が重解をもち、その重解が正の実数であるとき、定数 p の値の範囲を求めよ。

――――――  
これら10題は、因数分解・平方完成・解の公式の使い分け、判別式による解の性質の判断、具体的事象（図形・仕事算・運動）への応用、関数の最大値・最小値など、多面的に二次方程式の理解と活用を問う構成になっています。
